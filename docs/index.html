              勉強は「才能」じゃなく仕組みで勝てる。

              このページは、受験勉強を「回る形」に整える入口。
              迷ったら 全体設計 → 毎日の型 → ミス対策 の順で進める。

              まず読む順番（最短で効く）

              • 全体設計（やること／捨てること）
                → 全体設計：やること／捨てること

              • 毎日ルーティン（60分の型）
                → 毎日ルーティン：60分

              • ミスの5分類（原因を潰す）
                → ミスの5分類と対策

              このページだけでできる：今日の30分

              目的：迷いを減らして、明日も続く形にする。

              1. 5分：今日やる範囲を1つに絞る

                • 「やらない単元」も1つ決める（迷いが減る）

              2. 20分：問題を解く（役割はミス発見）

                • 解けた/解けないより「どこで落としたか」を見る

              3. 5分：ミスを1つだけ修正する

                • 原因1行：＿＿＿＿
                • 次の対策1行：＿＿＿＿
                • 明日やる問題：＿＿＿＿

              目的別：今すぐ選ぶ

              • 全体がぐちゃぐちゃ → 優先順位（伸びる順の決め方）
              • 毎日続かない → 毎日ルーティン：60分
              • 時間配分が不安 → 時間配分の目安
              • 暗記がつらい → 思い出す暗記（Recall）
              • 復習が回らない → 復習の流れ（回るシステム）
              • 伸びない原因が不明 → ミスの5分類と対策

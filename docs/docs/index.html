              オンライン自習室
              勉強は「才能」じゃなく
              仕組みで勝てる

              ここは受験の勉強法を、迷わず実行できる形にまとめる場所。
              「がんばる」より先に、毎日回る型を作ろう。

              最優先：全体設計へ 毎日の型（ルーティン）へ ミス分析と対策へ
              優先度
              全体設計 → 毎日の型
              主役
              ミス修正（再発防止）
              基本
              スマホ前提で回す
              目的別：今すぐ選ぶ
              「困ってること」から入れば迷いが消える。まずは1つだけでOK。
              全体がぐちゃぐちゃ 毎日続かない 伸びない原因が不明 暗記がつらい 復習が回らない
              まず読むべき3本（最短で効く順）
              順番どおりに読むと、あなたの勉強が“回り始める”。
              ①
              全体設計（やること/捨てること）
              伸びない人は「努力不足」より優先順位が崩れてることが多い。 まずは科目・単元・時間配分を“設計”して、迷いを消す。
              読む → 迷いが消えて、勉強が軽くなる
              ②
              毎日の型（60分/120分/週末）
              勉強が続かない原因は、やる気じゃなく判断コスト。 “毎日やる順番”を固定して、実行を自動化する。
              読む → 継続が安定して、ミスが溜まらない
              ③
              ミスの5分類と対策（再発防止が主役）
              問題を解く目的は“量をこなす”じゃない。ミス発見のため。 原因を分類できるほど、修正が速くなる。
              読む → 同じミスが減って、点が伸びる
              このサイトのルール（最重要）
              • 問題は主役“っぽく見える”が、本当の主役は「再発防止」（ミスを潰した数が伸びを決める）
              • 分類は1つじゃない（強化段階・理解度・教科でやり方を変える）
              • 時間の訓練は、普段の勉強でやりすぎない（必要な時期に“過去問実習”で固める）
              ※学校の成績の取り方とは切り分けて設計（受験向け）。
              次に伸ばす領域（順番）
              あなたの状態に合わせて、読む順番を変えていい。迷うなら上から。
              記
              暗記のやり方
              暗記は“読む”より思い出す訓練。回し方と負荷調整がすべて。
              読みどころ → 忘却を前提に設計する
              復
              復習のやり方
              復習は“時間をかける”ほど良いわけじゃない。再発防止の最短手順を固定する。
              読みどころ → ルーティンに埋め込む
              ミ
              ミス分析と対策
              ミスの種類が分かれば、修正は“テンプレ化”できる。伸びる人はここが速い。
              読みどころ → 原因ラベルを固定する

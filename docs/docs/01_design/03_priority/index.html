              原則
              やることが多い人ほど「順番」で勝つ
              勉強は量より配列。順番が正しいほど、同じ時間で伸びる。

              優先順位の固定（これでOK）

              1. 頻出×取れる
              2. ミスが多い
              3. 暗記で即伸びる
              4. 理解を足す（必要分だけ）
              コツ

              「今日は何しよう…」をゼロにする。前日に次を決めておく。

    • 勉強の基本設計
      • 暗記と復習
        • 復習のやり方
          • ミス分析と対策
            • 毎日のルーティン
              時間配分の目安（問題中心で伸ばす）
              • 基本比率（迷ったら固定）
              • 1日の中でのおすすめ配分（例）
              実用
              時間配分は「問題の比率」で決まる
              理解の時間を削るのではない。問題で判定して、必要分だけ理解を足す。

              基本比率（迷ったら固定）

              • 平常期：理解 3：問題 7
              • 直前期：理解 1：問題 9
              判断基準

              問題が解けない理由が「知識不足」なら暗記へ。
              「手順が崩れる」なら解き方を固定。
              「理解不足」なら短い解説→すぐ問題。

              1日の中でのおすすめ配分（例）

              • 最初：問題（判定）
              • 次：ミス分類
              • 最後：復習（短く強く）
              今日のチェック
              • 問題の時間が一番長い
              • 解説を読み続けず「次の1問」に戻れた

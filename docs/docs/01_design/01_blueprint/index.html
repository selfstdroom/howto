              完成版
              受験勉強は「本番で解ける」を作る作業
              やることは多い。でも順番と型を固定すれば、迷いは消える。ここだけで全体が回る。
              結論（これだけ）

              伸びる人は「問題 → ミス分類 → 復習」を毎日回している。理解は必要だが、理解だけでは点にならない。“正解できる状態”を作るのが勉強。

              ゴール
              「説明できる」ではなく「本番で解ける」
              解ける状態＝同じミスをしない／時間内に処理できる／再現できる
              勝ち筋
              問題時間を最大化する
              理解は短く、問題で判定し、ミスを潰す。これが最短。

              1) 勉強の基本ループ（毎日これ）

              • 問題を解く（最初から満点狙いじゃなくてOK。判定のために解く）
              • ミスを分類（理由を“言語化”してラベルを貼る）
              • 復習で再発防止（次回の正解率を上げる最小作業だけする）
              重要：復習は“納得”じゃない

              復習の目的は「分かった気」ではなく、次に同じ問題を落とさない状態を作ること。だから復習は短くていい。

              2) 優先順位（伸びる順番）

              これが崩れると一生迷う。
              基本はこの順番。

              1. 頻出×取れる問題（努力が点になりやすい）
              2. ミスが多い単元（改善余地が大きい）
              3. 暗記で即伸びる範囲（英単語・古文単語・公式・用語など）
              4. 理解系の詰め（必要分だけ）

              伸びない人は「好きな単元」や「気分」で順番が毎日変わる。

              3) 時間配分（迷ったらこれ）

              • 平常期：理解 3：問題 7
              • 直前期：理解 1：問題 9（演習と復習が主）
              今日の判定（YESなら伸びる日）
              • 勉強時間の半分以上を「問題」に使った
              • 間違いをその場で1つ“潰した”
              • 次にやる問題が決まっている

              4) 暗記の原理（最短）

              暗記は「書く量」ではなく、思い出す回数で決まる。

              • 覚える → すぐ小テスト（再生）
              • 間違える → その場で最小修正
              • 翌日 → もう一回テスト
              • 3日後/7日後 → まとめてテスト
              暗記は“作業”ではなく“試験”

              暗記はテスト形式が最強。ノートは補助。覚えたかは再生できたかで判定する。

              5) ミス分析（5分類だけで十分）

              ミスはこれだけに分ける。

              • 知識不足：覚えてない
              • 理解不足：理由や仕組みが弱い
              • 手順ミス：解き方の順序が崩れてる
              • 条件読み違い：問題文の処理が雑
              • ケアレス：計算/転記/符号/単位

              → 対策は「分類ごとの型」で固定できる（次の記事で完成テンプレを出す）。

              6) 1日60分の最小ルーティン（時間がない人用）

              • 5分：今日やる問題を決める（“選ぶ”で消耗しない）
              • 40分：問題を解く（解説読みは最低限）
              • 10分：ミス分類＋復習（再発防止だけ）
              • 5分：明日の問題を決める（次の自分を助ける）

              7) 最後に：このサイトの使い方

              • 左メニューからセクションを選ぶ
              • セクショントップにあるカード目次から記事へ
              • 各記事の最後にチェックがあるので、毎日1つだけでも回す

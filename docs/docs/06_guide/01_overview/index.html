              A. 結論（迷わない言い切り）

              この教材は「読む」よりも、毎日 “同じ手順” で回してミスを減らすために使います。 最初の10分でやることはこれだけです。

              1. 読む順番を固定（最初に読むページを決める）
              2. 毎日の型を1つだけ決める（60分でOK）
              3. 復習の流れを1本化（ミス→原因→修正→再テスト）

              これで、今日からブレません。

              B. なぜそうするのか（理屈：迷いを消す）

              1. 判断回数を減らすほど続く 「今日は何やろう…」の迷いが増えると、勉強は“スタート”で止まります。順番と型を固定して、迷いを消します。

              2. ミスは “見つけた瞬間” が一番価値が高い 問題は目的ではなく、ミスを見つける道具。見つけたミスを放置すると、次も同じ場所で落ちます。

              3. 復習は “回数” が勝つ たまに長時間より、短くても毎日「ミス→修正→再テスト」を回したほうが伸びます。

              4. スマホ前提で運用すると、日常に溶ける 机に座れない日があっても、最低限の流れが維持できます。

              C. 今日の手順（まず10分：セットアップ）

              0〜2分：読む順番を決める（固定する）

              次の3つだけ、今開いてください（ブックマーク推奨）。

              • 全体設計（やること／捨てること）
              • 毎日の型：60分
              • 復習フロー（ミス→修正→再テスト）

              2〜6分：今日の型を1つ決める（迷いゼロ）

              「毎日の型：60分」から、今日の自分に合う形を1つだけ選びます。 選んだら、1週間はそれを固定してください（気分で変えない）。

              6〜10分：復習メモのテンプレを用意する（スマホでOK）

              紙でもスマホでもOK。1つの型に揃えます。コピペで使ってください。

              【今日のミス】（最大3つ）
1) ミス：＿＿＿＿＿＿＿＿
   原因：＿＿＿＿＿＿＿＿（知識不足／読み違い／手順ミス／計算ミス／時間配分…）
   修正：＿＿＿＿＿＿＿＿（次はこうする）
   再テスト：いつ？（今日／明日／週末）

2) …

【明日の最初にやること】（1行）
＿＿＿＿＿＿＿＿＿＿

              D. 具体例（よくあるケースで）

              例1：無料版で「何から読めばいい？」となる

              • まずは 全体設計→毎日の型→復習フロー の3点だけ固定
              • そのあとに、苦手に応じて分岐（暗記／ミス分析）

              次に読む：

              • 思い出す練習（暗記の中核）
              • ミスの種類（原因の切り分け）

              例2：自習室参加で「報告が苦手」

              • 報告は“長文”じゃなくてOK
              • ミス3つ＋明日の最初の1行 だけで十分に価値が出る

              E. よくある誤解（最低3つ）

              1. **「読むだけで伸びる教材」**ではない 読むのは入口。伸びるのは 毎日同じ手順で回したとき。

              2. **「問題をたくさん解くほど正義」**ではない 問題はミス発見の道具。ミスを放置したまま量を増やすと、同じ落とし穴が増えるだけ。

              3. 「復習＝もう一回解く」だけではない 復習は 原因→修正→再テスト までが1セット。

              F. チェックリスト（YES/NOで自己診断）

              • YES / NO：今日やる順番が固定されている
              • YES / NO：毎日の型（60分など）が1つ決まっている
              • YES / NO：ミスは最大3つに絞って記録している
              • YES / NO：ミスごとに「次はこうする」が書けている
              • YES / NO：再テスト日が決まっている（今日/明日/週末）

              YESが3つ未満なら、まずは型の固定に戻ってOKです。

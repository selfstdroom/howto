              毎日ルーティン
              時間がない人ほど「毎日の型」で勝つ
              伸びる人は、長時間やる日より「崩れない日」を作っている。ルーティンはそのための道具。
              結論（ルーティンの核心）

              毎日やる内容を固定すると、迷いが減って勉強が続く。
              ルーティンは「やる気」を増やすものじゃない。毎日の判断回数を減らして、勉強を回す仕組み。

              まず最初に：このページで言う「ルーティン」の意味

              このページのルーティンは、「気合いで頑張る予定表」じゃない。
              毎日、最低限これだけ回せば点が伸びるという 最小構成 のこと。

              • ルーティンがある → 迷わない → 着手が速い → 続く
              • 続く → ミスが見える → ミスが減る → 点が上がる
              よくある勘違い

              「問題が主役」＝「解くことが目的」ではない。
              問題を解くのは、ミスを見つけるため。ミスが見えない勉強は、点が伸びにくい。

              なぜルーティン化すると強いのか（理屈）

              受験勉強が崩れる原因は、能力不足よりも毎日の判断コストが多いこと。

              • 「今日は何やろう…」で時間が溶ける
              • 決めた後も「これでいいのか？」が残って集中が切れる
              • 結果、問題に触れる時間が減る（＝ミスが見えない）

              ルーティン化はこれを消す。

              • やることが固定 → 判断が減る
              • 判断が減る → 着手が速くなる
              • 着手が速い → 毎日続く
              • 続く → ミスが減って点が伸びる
              ルーティンのゴール

              「完璧な計画」じゃない。毎日回る最小構成を作ること。

              まず決めるルール（ここだけは固定）

              1. 毎日やる時間を固定（15分でもOK。毎日が最優先）
              2. 内容は「問題 → ミス → 復習」の順に固定
              3. 翌日の問題を前日に決める（迷いゼロ）
              ここがズレると崩れる
              • 「理解だけの日」が増える（ミスが見えない）
              • 復習が長引いて時間が足りなくなる（原因が曖昧）
              • その日の気分で科目が変わる（積み上がらない）

              今日から使える：60分ルーティン（時間がない日の型）

              • 5分：今日やる問題を確定（前日に決めていれば0分）
              • 40分：問題（目的：ミス発見）
              • 10分：ミス分類＋復習（再発防止だけ）
              • 5分：明日の問題を決める（次の自分を助ける）
              60分ルーティン：守るポイント
              • 「理解だけ」で終わらせない（必ず問題で締める）
              • 復習は10分で切る（長引くなら“原因の言葉”が曖昧）
              • 最後の5分で「明日の問題」を決める（継続の要）

              120分ルーティン（標準：伸ばしたい日の型）

              • 10分：今日の問題セットを確定（優先順位で選ぶ）
              • 75分：問題（演習：ミスが出る量を確保）
              • 20分：ミス分類＋復習（原因1行＋対策1行＋再テスト1問）
              • 15分：暗記（間違いだけを思い出す）
              120分のコツ

              伸びる日は「問題の量」より、ミスを潰した数で勝つ。復習を削らない。

              重要：時間の練習は「毎日」じゃなく別枠でやる

              普段の勉強で優先するのは、再現性（同じ手順で解ける）。
              時間配分の実戦練習は、過去問などでまとめてやる方が効率がいい。

              • 普段：手順と判断を固定して「ミスの種類」を減らす
              • 別枠（過去問）：時間配分・取捨選択・本番のテンポを練習する
              理由

              普段から時間に追われると、ミスの原因が見えにくくなる。
              まずはミスの修正で土台を固めて、時間練習は「実戦枠」でまとめてやる。

              週末ルーティン（まとめて整える日）

              平日を回すほど、週末の“修理”が効く。

              • 30分：1週間のミスを集計（多い順に並べる）
              • 60分：ミスが多い単元の同類題を解く（弱点を露出させる）
              • 30分：暗記の総テスト（チェックだけ）
              • 15分：来週の「毎日やる範囲」を固定
              週末にやると一気に伸びる作業
              • ミスが多い原因（例：手順ミス）を“手順テンプレ”で統一する
              • 暗記は「全体」じゃなく「間違いだけ」を集中的に
              • 来週の“毎日枠”を固定しておく

              ルーティンが崩れる典型 → 直し方

              崩れる原因
              毎日やる内容が多すぎる
              直し方：60分版に戻して「問題 → ミス → 復習」だけ残す。
              崩れる原因
              その日の気分で科目が変わる
              直し方：前日に“明日やる問題”を決めて固定。
              崩れる原因
              復習が長引いて疲れる
              直し方：原因の言葉をつけ直す。対策は1行、再テスト1問で切る。
              崩れる原因
              スマホが気になって集中が切れる
              直し方：置く/置かないではなく「役割」を固定する（タイマー／記録／必要な解説確認）。通知は切る。

              スマホ前提の“最小運用”（机にスマホがあっても成立）

              スマホがあるとダメ、じゃなくて役割を固定する。

              • タイマー（40分/75分）
              • 復習メモ（原因1行＋対策1行）
              • 解説確認（必要なときだけ。だら見禁止）
              • 通知はオフ（ここだけは必須）
              まとめ

              スマホは「置く/置かない」じゃなく、使い方を固定する。

              今日のチェック（YESが増えるほど強い）

              チェック
              • 今日やる問題が決まっている
              • ミスが1つ見つかった（理由まで言える）
              • ミスを1つ潰した（原因1行＋対策1行＋再テスト）
              • 明日の問題が決まっている

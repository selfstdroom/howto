              完成版
              復習は「長くやる」より「次に正解する形」を作る
              復習のゴールは“理解した気”じゃない。次に同じ形が出たとき、時間内に正解できること。
              結論（復習はこれだけ）

              復習は「原因ラベル → 直し方1行 → すぐ再テスト」の3点セット。
              これを毎回同じ手順で回すと、復習がブレなくなる。

              なぜ「短く強く」が正しいのか（理屈）

              復習が長くなる原因はだいたい2つ。

              1. “安心”のために読み続ける（理解した気になる）
              2. 何をすれば直るかが曖昧（だから時間をかけてしまう）

              でも受験では、点に直結するのは「再現できること」。
              つまり、復習の中心は「読む」ではなく“思い出す／解く”に寄せた方が強い。

              • ただ読む：本番の動作と違う（本番は見ずに解く）
              • すぐ再テスト：本番の動作に近い（再現性が上がる）
              復習が効く条件

              復習が効いている状態＝「次に同じ形が出たら、何をして正解するか」が決まっている状態。

              まず整理：「理解不足」と「手順が曖昧」は別

              ここを混ぜるから、復習がグダる。

              理解不足
              なぜそうなるか／理由が弱い
              例：公式の意味、現象の仕組み、証明の流れが説明できない
              手順が曖昧
              どう解くか／順序が固定されてない
              例：問題を見るたびにやり方が変わる、途中で迷う、計算の順番が毎回違う

              理解できていても、手順が固定されていないと本番で崩れる。
              逆に、手順が固定されると、理解が浅くても点が伸びる場面が多い（特に典型問題）。

              復習テンプレ（これを毎回コピペ運用）

              • 原因ラベル：知識／理解／手順／読み違い／ケアレス
              • 原因1行：「なぜ落ちたか」を1行で言う（長文禁止）
              • 直し方1行：次に同じ形なら何をするか（行動で書く）
              • すぐ再テスト：同じ問題 or 同類題を1問、見ずに解く
              コツ

              「原因」と「直し方」は1行。長くなるほど実行できない。
              直し方は「気をつける」禁止。手順に落とす（例：最後に条件を線引きして再確認）。

              ラベル別：直し方の型（ここだけ覚えればOK）

              1) 知識不足（覚えてない）

              • 直し方：再生テスト（隠して言う／小テスト）
              • 再テスト：翌日にもう一回（ここが大事）

              2) 理解不足（理由が弱い）

              • 直し方：“理由を1文”にする（教科書から拾って1文化）
              • 再テスト：同類題を1問すぐ解く（読むだけで終わらない）

              3) 手順が曖昧（解き方がブレる）

              • 直し方：解き方を3手に固定（順番が命）
              • 再テスト：同類題を時間を測って解く

              4) 読み違い（条件処理が雑）

              • 直し方：条件に線→条件を箇条書き→最後に条件に戻る
              • 再テスト：同類題で「条件処理だけ丁寧に」やる

              5) ケアレス（符号・転記・単位など）

              • 直し方：見直し箇所を固定（毎回同じ場所を見る）
              • 再テスト：見直しチェック込みで解き直す

              具体例（実際の復習メモの形）

              例：数学（手順ミス）
              原因1行
              置換したあとに「元の変数に戻す」を忘れて式が崩れた。
              直し方1行
              置換したら最後に「戻す」チェックを必ず入れる（解答欄の直前）。
              例：英語（知識＋読み違い）
              原因1行
              that節を見落としてSVの区切りがズレた。
              直し方1行
              接続詞・関係詞を○で囲み、節のカタマリごとに線を引く。

              どれくらい時間をかける？（目安）

              • 復習メモ：1問あたり1〜3分
              • 再テスト：1問（または同類題1問）
              • それ以上やるのは、基本的に「原因が曖昧」なサイン
                → まずラベルを決め直す

              スマホで読む人向け：最小チェック（今日やること）

              YESなら今日の復習は勝ち
              • 原因ラベルをつけた（知識/理解/手順/読み/ケアレス）
              • 原因と直し方が“1行”で書けた
              • 見ずに1問やり直した（ここまでが復習）
